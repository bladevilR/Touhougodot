# 农场系统集成测试指南 (Integrated Farming Test)

现在农场系统已经完全集成到游戏世界中，玩家可以操控角色（藤原妹红）进行种田。

## 1. 运行测试场景

在 Godot 编辑器中打开并运行：
`scenes/farming/FarmingTestScene.tscn`

## 2. 操作说明

### 移动
- **WASD / 方向键**: 移动角色。

### 农具切换 (键盘数字键)
- **[1] 手**: 收获作物。
- **[2] 锄头**: 耕地（对着绿色草地使用）。
- **[3] 水壶**: 浇水（对着褐色耕地使用）。
- **[4] 肥料**: 施肥。
- **[5-8] 种子**: 
    - [5] 番茄 (春/夏)
    - [6] 小麦 (夏/秋)
    - [7] 南瓜 (秋)
    - [8] 红萝卜 (春/秋/冬)

### 交互
- **鼠标左键**: 对着高亮的方框点击，使用当前工具。
- **空格键 (Space)**: 快速跳过一天（测试用，观察生长）。

## 3. 视觉反馈
- 当你靠近农田时，鼠标指向的格子会出现**黄色高亮框**。
- **绿色方块**: 未开垦的草地。
- **浅褐色方块**: 已开垦的干旱土地。
- **深褐色方块**: 已浇水的土地。
- **作物**: 会显示从小芽到成熟花朵的生长过程。

## 4. 常见问题
- **无法种植？** 必须先用锄头[2]把地耕了。
- **作物不长？** 必须每天用水壶[3]浇水，然后按空格过夜。
- **种子没反应？** 检查屏幕左上角的季节，有些作物只能在特定季节种植。
