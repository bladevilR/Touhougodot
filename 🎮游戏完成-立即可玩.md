## 🎯 现在就能玩！

### 第1步：打开Godot
1. 启动 **Godot 4.5** 编辑器
2. 点击 "导入"
3. 选择 `touhou-godot` 文件夹
4. 打开项目

### 第2步：直接运行
1. 按 **F5** 键（或点击右上角播放按钮）
2. 游戏启动！

### 第3步：开始玩！
- **WASD 或 方向键** - 移动
- **自动射击** - 靠近敌人就会自动攻击
- 击杀敌人获得经验
- 升级后血量回满

## 🔧 可选改进

### 如果想切换角色
1. 打开 `Player.tscn`
2. 在检查器中找到 `character_id`
3. 修改值：
   - 0 = 灵梦（博丽符纸）
   - 1 = 妹红（凤凰羽衣）
   - 2 = 魔理沙（星符）
   - 3 = 咲夜（银制飞刀）
   - 4 = 尤魔（刚欲汤勺）
   - 5 = 恋（本我地雷）

### 如果想添加更多武器
在游戏运行时，在控制台输入：
```gdscript
SignalBus.weapon_added.emit("star_dust")  # 添加星符
SignalBus.weapon_added.emit("molotov")    # 添加鸡尾酒瓶
```

### 如果想替换临时图标
1. 准备角色/敌人图片
2. 在场景编辑器中选择 Sprite2D
3. 在检查器中更换 Texture
